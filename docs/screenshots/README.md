# Screenshot Guide

Drop your screenshots into this folder using these exact file names, and they will appear in the main README automatically.

| File name | What to capture |
|---|---|
| `banner.png` | Wide hero image — e.g. the booking map with the app visible (landscape, ~1280x400) |
| `passenger-booking.png` | Booking panel with pickup + destination set on the map |
| `passenger-fare-estimate.png` | The fare breakdown card before confirming a ride |
| `passenger-live-tracking.png` | An accepted ride with the driver marker and route line visible |
| `passenger-place-search.png` | Destination search dropdown showing OSM suggestions |
| `passenger-goods-booking.png` | "Travelling with goods?" toggled on with weight/volume filled |
| `ride-chat.png` | The in-ride chat between passenger and driver |
| `assistant-search.png` | AI assistant showing place results (e.g. "nearest pharmacy") |
| `assistant-book-ride.png` | Assistant result card with the "Book ride here" button |
| `driver-requests.png` | Driver home with a pending request and the "Preview" route button |
| `driver-live-route.png` | Driver's live route line to pickup after accepting |
| `driver-earnings.png` | Driver earnings dashboard |
| `admin-dashboard.png` | Admin dashboard stats page |
| `admin-live-ops.png` | Live Ops map with active rides/drivers |
| `admin-pricing.png` | Pricing configuration page |
| `admin-driver-approval.png` | Driver approval list |
| `admin-rides.png` | Ride management table |
| `android-home.png` | The APK running on a phone — home screen |
| `android-booking.png` | Booking a ride inside the Android app |
| `android-gps-permission.png` | Android's native location permission prompt |
| `aws-deployment-diagram.png` | Optional — a diagram/screenshot of the AWS console (EC2 instance or architecture drawing) |

Tips:
- Phone screenshots look best at their native portrait resolution.
- For desktop shots, a ~1280px-wide browser window keeps text readable in the README tables.
- PNG format, no need to resize — GitHub scales them to fit the table cells.
